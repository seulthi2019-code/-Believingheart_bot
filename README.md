# بوت "قلب سليم" — دليل التشغيل

## الملفات
- `bot.py` — الكود الرئيسي للبوت
- `requirements.txt` — المكتبات المطلوبة

## التشغيل على جهازك (لتجربة سريعة قبل الاستضافة)

1. تأكد أن لديك Python مثبتًا على جهازك (إصدار 3.10 أو أحدث).
2. افتح الطرفية (Terminal / cmd) داخل مجلد المشروع.
3. ثبّت المكتبات المطلوبة:
   ```
   pip install -r requirements.txt
   ```
4. ضع التوكن والمفتاح كمتغيرات بيئة قبل التشغيل:

   على ويندوز (cmd):
   ```
   set TELEGRAM_TOKEN=ضع_التوكن_هنا
   set GEMINI_API_KEY=ضع_المفتاح_هنا
   ```

   على ماك/لينكس:
   ```
   export TELEGRAM_TOKEN=ضع_التوكن_هنا
   export GEMINI_API_KEY=ضع_المفتاح_هنا
   ```

5. شغّل البوت:
   ```
   python bot.py
   ```

6. افتح تليجرام وابحث عن @Believingheart_bot وابدأ المحادثة.

ملاحظة: هذه الطريقة تعمل فقط طالما جهازك مفتوح ومتصل بالإنترنت. للحصول على بوت
يعمل 24 ساعة دون الحاجة لإبقاء جهازك مفتوحًا، انتقل إلى خطوة الاستضافة على Render
(سأشرحها لك في الخطوة التالية من المحادثة).

## الأمان
- لا تشارك التوكن ولا مفتاح Gemini مع أي شخص.
- لا ترفع هذين المفتاحين مباشرة داخل الكود إلى أي موقع عام (مثل GitHub) بصيغة نص واضح؛
  استخدم متغيرات البيئة كما هو موضح أعلاه أو خاصية "Environment Variables" في الاستضافة.
